Genetic Algorithms Learning Helper Toolbox

This project has beed discontinued This project originally was designed for Genetic Algorithms course in SBUK university. The aim of this project was to help students easily implement genetic algorithms for different problems by giving them a template and visualize the generations to see exactly what is going on. It was implemented using C++/Qt and for visualization it uses qcustomplot. Students can add any problem and it's solution as a separate class to the program and then choose it from a dropdownbox in the gui app, set it's parameters and see th generations progress. This project is in its early development phase.
